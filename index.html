How does the web work?

The web browser :

The web browser connects to the web server and sends an http request via the protocol stack for the desired page. if the page exists, the web server sends it. if the server cannot find the requested page, it will send an http 404 error message.

What do you need to be a web developer?

Web developers :

Web developers must develop a set of technical skills to create website. They include the markup language html, programming languages like javascript and sql, and multimedia publishing tools like Flash. You most also develop specific workplace skills to become a web developer.
